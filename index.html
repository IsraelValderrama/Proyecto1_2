1. Editamos el archivo de configuración personalizada de GRUB:
sudo nano /etc/grub.d/40_custom
  1. Agregamos las siguientes líneas al final del archivo:
set superusers="root"
password root PruebaDebian123!
  1. Guardamos y cerramos el archivo.
CTRL ^O para guardar.
CTRL ^x para salir.

  1. Generamos una contraseña cifrada:
sudo grub-mkpasswd-pbkdf2
  1. Copiamos el hash generado y editamos nuevamente el archivo 40_custom:
sudo nano /etc/grub.d/40_custom
  1. Reemplazamos la línea de la contraseña por: 

sudo chmod 700 /etc/grub.d/40_custom
  1. Editamos el archivo de configuración de GRUB:

sudo nano /etc/default/grub
  1. Cambiamos la línea ‘GRUB_TIMEOUT=5' por:
    GRUB_TIMEOUT=0
  1. Guardamos y cerramos el archivo.
CTRL ^O para guardar.
CTRL ^x para salir.

Después de que hayamos realizado los cambios en la terminal debemos de poner este comando:

sudo update-grub
